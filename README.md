<h1 align="center">Kanban Board</h1>

<p>full-stack web app built with <b>React</b>, <b>TypeScript</b> and <b>DndKit</b> on the front-end, <b>Node</b> and <b>Express</b> on the back-end and <b>MongoDB</b> as the database.</p>

<h2 align="center">The Board</h2>
<div align="center"> 

![Adding/Removing Workouts Gif](https://i.imgur.com/okM4x6c.gif)
<p>A classic kanban board with Drag & Drop of tasks using the dnd kit library; The App is strongly typed with clear payload contracts and clean separation between UI and domain logic. Every data changed by the user persists in the database.</p>


<h2 align="center">Authentication</h2>
<div align="center"> 
  
![Workout Auth Gif](https://i.imgur.com/Ezf8cc8.gif) 

<p>The authentication system was made using <b>JSON Web Token (JWT)</b> and <b>local storage</b> to store the sessions and keep it simple. </p>
